# デザインモダナイゼーション実装計画

## ゴール

ユーザーが「WOW」と感じるような、モダンでプレミアムなデザインに刷新する。
既存の「うづきちゃん（ウサギ）」の可愛らしさを残しつつ、洗練されたUIを実現する。

## ユーザーレビューが必要な事項

- [ ] 配色パレットの変更（ラベンダー基調は維持しつつ、よりモダンなグラデーションなどを導入して良いか）
- [ ] Google Fonts の導入（日本語フォント: `M PLUS Rounded 1c` 推奨）
- [ ] ボタン配置の微調整（使いやすさ向上のため）

## 変更内容

### STATIC (CSS/Images)

#### [MODIFY] [style.css](file:///c:/Users/onoda/Desktop/april_AWS/rabbit_Java/static/style.css)

- **CSS変数の導入 (`:root`)**: 色、フォント、スペーシングを一元管理。
- **リセットCSSの強化**: ブラウザ間の差異をなくす。
- **タイポグラフィ**: Google Fonts (`M PLUS Rounded 1c`) を適用し、可読性と親しみやすさを向上。
- **グラスモーフィズム**: カードやヘッダーにすりガラスのような効果 (`backdrop-filter`) を導入。
- **ボタン**: ホバーエフェクト、微細なアニメーションを追加。
- **レスポンシブ**: スマートフォンでも美しく見えるようパディングやフォントサイズを調整。

### TEMPLATES

#### [MODIFY] [base.html](file:///c:/Users/onoda/Desktop/april_AWS/rabbit_Java/templates/base.html)

- `head` タグ内に Google Fonts のリンク (`preconnect` 含む) を追加。
- ビューポート設定の確認。

## 検証計画

### 手動検証

1. **トップページ**:
   - ヒーローセクション（ウサギの画像）が美しく表示されるか。
   - フォントが適用されているか。
2. **日記一覧**:
   - グリッドレイアウトまたはカードリストが崩れていないか。
   - 日記カードのホバーアニメーション確認。
3. **詳細・編集・新規作成画面**:
   - フォームの入力しやすさ（フォーカス時の枠線色変更など）。
4. **モバイル確認**:
   - ブラウザのデベロッパーツールでスマホサイズにして崩れがないか確認。
