# Miruto風デザイン実装計画

## コンセプト

「Natural, Gentle, Modern」
既存のラベンダー・グラスモーフィズムから、より温かみのある**ナチュラル・ベージュ**基調のデザインへシフトします。
`miruto.shop` の特徴である「余白」「非対称な角丸」「優しい色使い」を取り入れます。

## 色彩計画 (Color Palette)

- **Base (背景)**: `#FAF7F4` (温かみのあるオフホワイト)
- **Text (文字)**: `#333333` (柔らかい黒)
- **Accent (強調)**: `#D88A8A` (くすみピンク/コーラル) ※または既存のラベンダーをこのトーンに合わせて調整
- **Secondary**: `#F0EBE5` (少し濃いベージュ、セクション背景用)

## フォント (Typography)

- **Noto Sans JP** (Google Fonts)
  - ウェイト: 300 (Light), 400 (Regular), 500 (Medium)
  - 文字間隔 (`letter-spacing`) を広めにとり、ゆったりとした印象を出す。

## 主な変更点

### 1. `base.html`

- フォント読み込みを `M PLUS Rounded 1c` から `Noto Sans JP`、およびアクセント用の欧文フォント（例: `Josefin Sans` や `Lato`）に変更。

### 2. `style.css` - デザインルール

#### レイアウト

- **余白**: セクション間のマージンを大きく取る (`80px`〜`100px`)。
- **コンテナ**: 以前より幅を広げつつ、左右の余白をしっかり確保。

#### UIパーツ

- **ボタン**:
  - 完全なカプセル型 (`border-radius: 50px`).
  - 枠線のみのゴーストボタンと、塗りつぶしボタンの使い分け。
  - ホバー時にふわりと浮くようなドロップシャドウ。

- **画像 / カード**:
  - **非対称な角丸**: 左上と右下だけ大きく角を丸める (`border-radius: 60px 0 60px 0`) など、有機的な形状に。
  - 枠線は廃止し、背景色との微妙なコントラストやソフトなシャドウで表現。

### 3. 各ページの調整

- **トップページ**: ヒーロー画像を画面幅いっぱいにしつつ、非対称な切り抜きを適用。
- **日記一覧**: グリッドレイアウトを維持しつつ、カードを「ポラロイド風」または「雑誌の切り抜き風」にアレンジ。

## ユーザーへの確認事項

- 元の「ラベンダー色（紫）」を維持しますか？それとも `miruto.shop` のような「コーラルピンク/ベージュ」に変更しますか？
  - **提案**: ベースはベージュ (`#FAF7F4`) にし、アクセントカラーとしてラベンダーを少し落ち着いたトーン (`#B39DDB` → `#957DAD`) で使う「ラベンダー・ナチュラル」な方向性を推奨します。
